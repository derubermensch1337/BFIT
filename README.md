# BFIT

Beer Fridge Inventory Tracking (BFIT) is a system that logs who takes drinks from a fridge, how much they consume, and provides statistics on current stock and usage.

This repository contains the embedded code, server/backend logic, documentation, and supporting files for the project.

## Repository structure
BFIT
├── docs/                 # Documentation (system design, specs)
├── src/                  # Firmware source code for Arduino
├── include/              # Header files
├── figs/                 # Figures for README, docs
├── platformio.ini        # Build and environment config
